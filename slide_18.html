✅ Примеры промптов
📋 Проведение FMEA
«Ты — инженер по качеству с опытом проведения FMEA. Проведи FMEA-анализ процесса контактной сварки кузовных деталей из низкоуглеродистой стали. Выяви не менее 5 потенциальных дефектов. Для каждого дефекта укажи: причины, последствия, текущие средства контроля. Рассчитай RPN (приоритетное число риска) по шкале 1–10 для каждого дефекта. Предложи корректирующие мероприятия для трёх дефектов с наибольшим RPN.»
🔍 Расследование дефектов по качеству
«Ты — инженер-технолог. Проведи расследование дефекта "выплеск" при контактной сварке. Используй данные из приложенного файла: журнал параметров сварки, протоколы испытаний, фото дефектов. Собери все факты, выяви временные и технологические закономерности. Составь отчёт по структуре: описание дефекта, условия возникновения, предварительные выводы, список необходимых дополнительных данных.»
❓ Причинно-следственный анализ (5 почему)
«Ты — инженер по качеству. Проведи анализ корневых причин дефекта "непровар" при контактной сварке с использованием метода "5 почему". На каждом шаге задавай вопрос "почему?" и давай ответ на основе имеющихся данных. В конце укажи корневую причину и предложи конкретные корректирующие мероприятия. Если данные неполные, укажи, какая информация требуется для завершения анализа.»
📊 Анализ логов сварки и статистики дефектов
«Ты — специалист по анализу данных. Проанализируй прикреплённый файл с логами сварки (параметры тока, времени, усилия) и статистику дефектов за последние 3 месяца. Выяви корреляции между отклонениями параметров и возникновением дефектов. Определи топ-3 параметра, которые наиболее сильно влияют на качество сварки. Построй график зависимости и предложи оптимальные диапазоны параметров для снижения брака.»
🛠️ Разработка превентивных мер
«Ты — инженер по качеству. На основе результатов анализа дефектов и выявленных корневых причин (приложен файл с данными) разработай перечень превентивных мероприятий. Для каждой меры укажи: описание, ожидаемый эффект (снижение брака в %), срок внедрения, необходимые ресурсы, ответственного. Оцени эффективность каждой меры и предложи план внедрения с prioritetom (A, B, C).»
⚠️ Управление рисками
«Ты — инженер-аналитик. Проведи оценку рисков для сварочного производства на основе исторических данных о дефектах и отказах оборудования (файл приложен). Идентифицируй не менее 10 рисков, для каждого укажи: описание, вероятность (1–5), тяжесть последствий (1–5), уровень риска (произведение). Сгруппируй риски по категориям (технологические, оборудование, человеческий фактор, материалы). Предложи план управления рисками с приоритетными действиями для рисков с высоким уровнем.»
