Техники промптинга: Few-shot, Chain-of-Thought, Tree-of-Thought

📋 Few-shot
Даёте модели 2-3 примера, она продолжает в том же стиле.
Применение: генерация документов по шаблону, анализ дефектов по классификатору
Пример: «Вот 3 примера отчётов по дефектам. Составь четвёртый по аналогии»
📋 FEW-SHOT (ТЕХНОЛОГИЧЕСКАЯ КАРТА)
Вот 3 примера технологических карт контактной сварки:
1. Карта 1: Деталь — кронштейн, материал — сталь 08пс, толщина 1.5 мм, ток 8000 А, время 0.8 сек, усилие 250 даН.
2. Карта 2: Деталь — усилитель, материал — сталь 09Г2С, толщина 2.0 мм, ток 9500 А, время 1.0 сек, усилие 300 даН.
3. Карта 3: Деталь — панель, материал — сталь 08пс, толщина 1.2 мм, ток 7500 А, время 0.7 сек, усилие 220 даН.
Составь карту 4 для детали 'лонжерон', материал — сталь 09Г2С, толщина 2.5 мм.
+60% точность генерации по шаблону с Few-shot
🔗 Chain-of-Thought
Просите модель рассуждать пошагово.
Применение: сложные расчёты, FMEA-анализ, причинно-следственный анализ
Пример: «Реши задачу пошагово. Сначала определи причины, потом последствия, потом приоритеты»
🔗 CHAIN-OF-THOUGHT (FMEA/PFMEA)
Проведи FMEA-анализ процесса контактной сварки кузовных деталей. Реши пошагово:
1. Шаг 1. Перечисли возможные дефекты (минимум 5).
2. Шаг 2. Для каждого дефекта определи причины (минимум 3 на дефект).
3. Шаг 3. Для каждой причины оцени последствия (влияние на качество, безопасность, стоимость).
4. Шаг 4. Рассчитай приоритет (RPN) по формуле: тяжесть × вероятность × обнаруживаемость.
5. Шаг 5. Предложи корректирующие мероприятия для топ-3 причин с максимальным RPN.
−40% математических ошибок с Chain-of-Thought
🌳 Tree-of-Thought
Модель рассматривает несколько вариантов решения.
Применение: выбор оборудования, оптимизация процессов, оценка рисков
Пример: «Рассмотри 3 варианта решения. Для каждого укажи плюсы, минусы, риски. Выбери оптимальный»
🌳 TREE-OF-THOUGHT (ВЫБОР ОБОРУДОВАНИЯ)
Нужно выбрать оборудование для сварки кузовных деталей. Рассмотри 2 варианта:
Вариант 1: Сварочный робот с 6 осями.
Вариант 2: Подвесные контактные клещи с ручной подачей.
Для каждого варианта укажи: производительность, точность, стоимость, сроки поставки, требования к персоналу, гибкость.
Потом сравни по критериям: QCD (качество, стоимость, сроки). Выбери оптимальный вариант с обоснованием.
+35% качество выбора с Tree-of-Thought
📄 Простая задача ➔ 📋 Few-shot
📊 Сложный анализ ➔ 🔗 CoT
🎯 Выбор варианта ➔ 🌳 ToT
💡 Практическая рекомендация
Комбинируйте техники. Например, Few-shot + CoT для генерации документа с анализом.
