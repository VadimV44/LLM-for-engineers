✅ Примеры промптов
💡 Генерация идей
«Ты — креативный инженер-технолог. Предложи 10 нестандартных идей по снижению себестоимости сварки кузовных деталей без потери качества. Для каждой идеи укажи: описание, ожидаемую экономию (в %), сложность внедрения (высокая/средняя/низкая), потенциальные риски. Отсортируй по соотношению эффективность/сложность.»
📚 Расширение экспертизы
«Ты — эксперт по контактной сварке. Расскажи о современных методах контроля качества сварных соединений (ультразвук, тепловизионный контроль, акустическая эмиссия). Сравни их по точности, скорости, стоимости оборудования и сложности интерпретации. Приведи примеры применения в автомобильной промышленности. Какие методы лучше всего подходят для мониторинга в реальном времени?»
🎓 Персонализированное обучение
«Ты — наставник по технологии сварки. Составь персонализированный план обучения для инженера-сварщика со стажем 3 года, который хочет углубить знания в области контактной сварки алюминиевых сплавов. Укажи: темы для изучения (по порядку), рекомендуемую литературу, практические упражнения и контрольные вопросы для самопроверки. Адаптируй план так, чтобы заниматься можно было 2 часа в день в течение 3 месяцев.»
🔄 Непрерывное совершенствование
«Ты — консультант по Lean-производству. Проанализируй текущий процесс сварки кузовных деталей (приложен файл с описанием). Выяви узкие места и потери (время ожидания, лишние перемещения, перепроизводство, дефекты). Предложи конкретные улучшения с использованием метода PDCA. Для каждого предложения укажи: проблему, корневую причину, мероприятие, ожидаемый эффект, срок реализации и способ контроля.»
🤖 Чат-бот для консультаций
«Ты — разработчик голосового помощника для цеха сварки. Опиши сценарий работы чат-бота для мастера участка: бот отвечает на вопросы по технологии сварки, подбирает режимы по материалам, даёт рекомендации по устранению дефектов. Приведи примеры диалогов: вопрос "Какой ток для стали 08пс 1.5 мм?" — ответ с расчётом и ссылкой на техкарту. Укажи, какие данные должны быть в базе знаний бота и как обновлять их.»
🌍 Лучшие мировые практики
«Ты — аналитик по техническому развитию. Найди 5 успешных кейсов внедрения AI в сварочном производстве за последние 2 года (в автомобильной промышленности). Для каждого кейса укажи: компанию, технологию, проблему, результат (цифры), ключевые факторы успеха. Сделай краткий обзор и сравнительную таблицу по эффективности. Какие решения можно адаптировать для нашего производства? Какие риски нужно учесть при адаптации?»
