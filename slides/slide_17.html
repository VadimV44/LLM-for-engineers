✅ Примеры промптов
🔍 Поиск и пояснение информации
«Ты — инженер-технолог. В прикреплённом файле — технический паспорт сварочного оборудования на 120 страниц. Найди все разделы, где указаны требования к охлаждению трансформатора. Извлеки из них: тип охлаждения, расход воды, минимальное давление, температуру на входе и выходе. Представь результат в виде краткой выжимки с указанием номеров страниц и пунктов документа.»
📊 Структурирование и сжатие данных
«Ты — инженер-аналитик. Извлеки из прикреплённого технического паспорта оборудования ключевые параметры: производительность, габариты, энергопотребление, масса, класс точности. Представь результат в виде таблицы с колонками "Параметр", "Значение", "Единица измерения", "Допуск", "Ссылка на пункт документа". Если параметр не указан, оставь ячейку пустой.»
🔗 Трассировка требований
«Ты — инженер по качеству. Установи соответствие между требованиями заказчика (приложен файл с ТЗ) и внутренними стандартами предприятия (приложены ГОСТы и СТО). Для каждого требования из ТЗ укажи: статус (выполнено / частично / не выполнено), ссылку на конкретный пункт стандарта (номер раздела, пункта), пояснение по соответствию и предложение по доработке, если требуется.»
📄 Сравнение версий документов
«Ты — специалист по документообороту. Сравни две версии технологической инструкции по сварке (приложены файлы: v1 и v2). Выдели все изменения в разделах "Параметры сварки", "Требования к оборудованию" и "Контроль качества". Представь результат в виде таблицы: "Раздел", "Было", "Стало", "Тип изменения" (добавлено/удалено/изменено), "Комментарий". Отметь, какие изменения критичны для производства и требуют переобучения персонала.»
🌐 Перевод и адаптация документов
«Ты — профессиональный технический переводчик с глубоким знанием машиностроения. Переведи прикреплённый паспорт на оборудование с китайского на русский язык. Сохрани точную структуру документа: все заголовки, подзаголовки, списки, номера страниц, сноски, специальные обозначения и таблицы. Технические термины и торговые марки (если они на латинице) оставь без перевода. Единицы измерения и числовые значения сохрани в оригинальном виде. Аббревиатуры при первом упоминании расшифруй в скобках. Стиль — строгий, официально-деловой. Без комментариев и интерпретаций.»
📈 Обработка сводных таблиц
«Ты — инженер-аналитик. В приложенном файле — сводная таблица с параметрами сварки (ток, время, усилие, температура) и результатами контроля качества (количество дефектов, тип дефекта) за последние 6 месяцев. Преобразуй данные: сгруппируй по месяцам и типам дефектов, рассчитай средние значения параметров, построй сводную таблицу с колонками: "Месяц", "Тип дефекта", "Количество", "Средний ток", "Среднее время", "Среднее усилие". Отсортируй по убыванию количества дефектов. Выяви топ-3 комбинации параметров с наибольшим числом дефектов.»
📚 Управление опытом и знаниями
«Ты — инженер-методист. На основе приложенного архива технических отчётов, актов расследований дефектов, протоколов испытаний и технологических инструкций создай базу знаний по дефектам контактной сварки. Структурируй информацию по следующим категориям: тип дефекта, причины возникновения, методы обнаружения, корректирующие мероприятия, превентивные меры. Для каждого дефекта укажи ссылки на соответствующие документы. Предложи структуру базы знаний в виде иерархии разделов и подразделов для удобного поиска.»
