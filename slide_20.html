✅ Примеры промптов
🧮 Расчёт технологических параметров
«Ты — инженер-технолог. Рассчитай режимы контактной сварки для соединения листов стали 08пс толщиной 1.5 мм и 2.0 мм. Исходные данные: тип сварки — точечная, материал электродов — хром-циркониевая бронза. Определи: диаметр электрода, сварочный ток, время импульса, усилие сжатия, шаг между точками. Представь результат в виде таблицы с формулами расчёта.»
📐 Интеграция с CAD/CAE
«Ты — инженер-конструктор. На основе технического задания с параметрами (габариты 1200×800×600 мм, материал — сталь 09Г2С, толщина стенки 3 мм, нагрузка 5 т) сгенерируй базовую CAD-модель сварочного стола. Укажи ключевые размеры, типы соединений (сварные, болтовые), ориентировочную массу. Предложи проверочный расчёт на прочность в CAE-системе: точки приложения нагрузки, закрепления, ожидаемые напряжения и коэффициент запаса.»
⚙️ Автоматизация рутинных операций
«Ты — инженер-программист. Напиши Python-скрипт для автоматической обработки логов сварочного оборудования (файлы в формате CSV). Скрипт должен: 1) считывать все файлы из папки, 2) извлекать параметры сварки (ток, время, усилие) для каждой точки, 3) сравнивать их с допустимыми значениями из эталонного файла, 4) формировать отчёт о несоответствиях в формате Excel. Добавь комментарии к каждой функции.»
📊 Макросы для Excel
«Ты — инженер-аналитик. Напиши VBA-макрос для Excel, который автоматически строит сводную таблицу по дефектам сварки. Исходные данные: столбцы "Дата", "Дефект", "Причина", "Количество". Макрос должен: 1) группировать данные по месяцам и типам дефектов, 2) строить столбчатую диаграмму для топ-5 дефектов, 3) рассчитывать процент брака по каждому месяцу, 4) выделять ячейки с браком >5% красным цветом. Код должен быть оптимизирован для больших файлов (>10 000 строк).»
📎 Работа с разнородными данными
«Ты — инженер по данным. У тебя есть три источника информации: 1) таблица Excel с параметрами сварки, 2) текстовые логи (TXT) с ошибками оборудования, 3) фотографии дефектов (JPG) с подписями в отдельном файле. Объедини данные в единую базу для анализа. Для каждого дефекта укажи: параметры сварки в момент возникновения, сообщение об ошибке из лога, описание дефекта с фото. Предложи структуру базы данных (поле, тип, описание) и пример SQL-запроса для выборки топ-3 причин брака по месяцам.»
💰 Расчёт себестоимости
«Ты — инженер-экономист. Рассчитай себестоимость изготовления кузовной панели. Исходные данные: материал — сталь 08пс, толщина 1.2 мм, расход материала 2.5 кг/шт, стоимость материала 85 руб/кг; сварочные электроды — 0.5 шт/изделие, 120 руб/шт; трудозатраты — 15 мин/изделие, 800 руб/час; электроэнергия — 2 кВт·ч/изделие, 7 руб/кВт·ч; амортизация оборудования — 150 руб/изделие; накладные расходы — 35%. Рассчитай полную себестоимость и точку безубыточности при выпуске 10 000 шт/месяц. Представь результат в виде структуры затрат (прямые и косвенные расходы) с диаграммой распределения.»
